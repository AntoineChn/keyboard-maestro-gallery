# Safari Gesture Macros

本系列动作由 3 组 Macros 组成。

1. Safari Gesture：基础功能
2. Safari Gesture Palette：高级功能
3. Safari Gesture Controller：根据是否外接鼠标，自动开启/关闭手势，以节省电量

## 基础功能

- 网页前进
- 网页后退
- 网页到顶
- 网页到底
- 前一个标签页
- 后一个标签页
- 呼出高级功能菜单
